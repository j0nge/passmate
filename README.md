# Passmate — 농구 동아리 스코어보드

8분 1쿼터, 두 팀씩 매치를 돌리는 농구 동아리용 웹 스코어보드.

## 동작 모델
- 팀 3개(A·B·C) → 각 쿼터마다 한 매치(예: A vs B)
- 「다음 쿼터 →」 누르면 round-robin으로 자동 순환 (AB → BC → CA → AB ...)
- 「이번 쿼터 매치업」 영역을 탭하면 두 팀을 자유롭게 변경 가능
- **페어링별 누적 점수**(A vs B 22:18 등)와 **쿼터별 매치 기록**을 함께 표시
- 「초기화」 누를 때까지 모든 점수가 누적 유지

## 기능
- 팀 2~8개 추가/삭제, 팀명 편집
- 쿼터 카운트다운 타이머 (기본 8분, 분/초 단위 변경 가능)
- 0초 도달 시 알람음 + 진동 + 자동 정지
- 점수판 탭 → 이번 매치 점수 +1
- 팀별 선수 등록, 선수 버튼 탭 → 팀 점수 +1 + 선수 누적 득점 +1
- −1 버튼 / 되돌리기(↶) 버튼
- 편집 모드: 선수 드래그로 팀 이동, 더블탭으로 이름 변경/삭제
- 쉬는 팀 카드는 자동 숨김 (편집 모드에선 표시)
- 새로고침해도 유지 (localStorage)
- 모바일/태블릿(iPad) 최적화

## 사용
- **일반 모드**: 점수판/선수 버튼 탭 → 점수 입력
- **편집 모드** (우상단 「편집」): 팀명·선수명 변경, 드래그로 팀 이동, 팀 추가/삭제
- **「다음 쿼터 →」**: 다음 매치업으로 자동 순환
- **매치업 영역 탭**: 이번 쿼터 두 팀 직접 선택
- **「초기화」**: 모든 점수·기록 리셋 (팀/선수 명단은 유지)

## 로컬 실행
```bash
npx serve .
```
또는 `index.html`을 브라우저에서 직접 열기.

## Vercel 배포
1. 이 폴더를 GitHub 저장소에 푸시
2. https://vercel.com → New Project → 저장소 선택
3. Framework Preset: **Other** (자동 감지)
4. Deploy

또는 Vercel CLI:
```bash
npm i -g vercel
cd 260509_passmate
vercel
```
